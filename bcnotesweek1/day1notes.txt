Program terminology:

-Function: a block of code; has to be called
-Parameter or argument
-Return Value: functions always return a value
-Body: The code inside the function
-Throws: errors that occur in the function and are passed back
-Function signature: How to call of function name vaue = functionName(parameter1);
-Comment text in a code file
-DocBlock: special comment that defines how the function will behave
-Type: data type (integer, string, array, object)
